# Classes in NyaScript 
 

* Class example:
    ```
        owo litterbox Counter ->
            owo litter(init, max) purrs this is the constructor purrs ->
                owo my num = init
                owo my max = max
            nyanya
            owo pat count() ->
                if my.num < my.max
                    incwease my.num
                nyanya  
                gib my.num
            nyanya
        nyanya
    ```
* Class instance example: ``owo mew counter = Counter(0, 10)``
