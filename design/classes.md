# Classes in NyaScript 
 
* ``litterbox``: class
    * Example:
            ```
                owo litterbox Counter ->
                    owo litter(init, max) purrs this is the constructor purrs ->
                        owo my num = init
                        owo my max = max
                    nyanya
                    owo pat count() ->
                        if my.num < my.max
                            incwease my.num
                        nyanya  
                        gib my.num
                    nyanya
                nyanya
            ```
* ``mew``: object (instance of a class, if you prefer)
    * Example: ``owo mew counter = Counter(0, 10)``
